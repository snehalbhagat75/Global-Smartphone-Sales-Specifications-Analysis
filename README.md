# Global-Smartphone-Sales-Specifications-Analysis
Smartphone Market Analysis Dashboard (Power BI)

Built an interactive Power BI dashboard analyzing 900+ smartphone models across multiple brands.
Created KPI cards for Rating, RAM, Storage, Battery Capacity, and Processor Speed.
Developed brand-wise sales and pricing visualizations to identify market trends.
Implemented filters and slicers for dynamic product exploration.
Generated insights on 5G adoption, operating systems, and processor performance.
